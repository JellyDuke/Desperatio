📌 1. 서비스 개요
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 실시간 주가 데이터와 기술적 지표(RSI, MACD, 이동평균선 등)를 활용하여
- '매수 / 매도 / 유지' 추천을 제공하는 룰 기반 AI 투자 보조 웹서비스입니다.
- Webflow 기반의 단일 페이지 UI + JavaScript + Finnhub API를 결합해 동작합니다.

📌 3. 기능 기획
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① 기술지표 기반 해석
- RSI < 30 → 매수 / > 70 → 매도
- MACD > Signal → 매수 / < Signal → 매도
- 이동평균선 정배열: 상승 추세 +1점 / 역배열: 하락 추세 +1점

② 종합 판단 로직
- 각 지표별 매수/매도 점수를 합산해 확률(%) 계산
- 예: 매수점수 3, 매도점수 2 → 매수 확률 60%, 매도 확률 40%

③ 출력 예시
- 하루: RSI 78, MACD 하락 → 과매수
- 한 달: 60-120-200일선 역배열 → 하락 추세
- 🧠 AI 종합 판단: 매수 확률 35%, 매도 확률 65%

④ 자산 분석 기능
- 입력: 종목/금액/수익률
- 출력: 비중, 리밸런싱 제안, 유사 섹터 종목 추천

⑤ 로컬 저장 기능
- 최근 분석한 종목 3개 저장 → 빠른 재분석 가능

⑥ 예외처리 UX
- 잘못된 티커: “종목을 찾을 수 없습니다”
- API 오류: “잠시 후 다시 시도해주세요”

① 페이지 구성 흐름 (단일 페이지 기준)

┌──────────────────────────────┐
│ [Header] 로고 + 네비게이션   │
├──────────────────────────────┤
│ [Hero 섹션]                  │
│ - 종목 입력창 (티커) + 버튼  │
├──────────────────────────────┤
│ [차트 섹션]                  │
│ - 1일 / 1주 / 1개월 차트 탭 │
│ - 캔들차트 및 지표 표시      │
├──────────────────────────────┤
│ [분석 요약 결과 섹션]       │
│ - RSI, MACD 해석 텍스트      │
│ - 정배열 여부, 확률 카드     │
├──────────────────────────────┤
│ [내 자산 분석 섹션]         │
│ - 종목 / 투자금 / 수익률 입력 │
│ - 비중 계산, 리밸런싱 제안    │
│ - 유사 섹터 종목 추천 포함    │
├──────────────────────────────┤
│ [로컬 저장 종목 보기]       │
│ - 최근 분석 종목 리스트       │
└──────────────────────────────┘

② 주요 컴포넌트 요약

- `symbol-input`: 종목 입력창
- `chart-view`: OHLC 차트 및 지표 탭
- `indicator-box`: 지표 해석 결과
- `summary-card`: 확률 + 신호 요약
- `portfolio-analyzer`: 자산 입력 및 분석
- `saved-symbols`: 최근 종목 로컬 리스트

③ Webflow 구조 가이드

- `section` 단위 div 구성
- `flexbox/grid` 사용
- 반응형 디자인 대응
- 각 섹션에 `id` 지정 가능 (앵커 네비게이션)
